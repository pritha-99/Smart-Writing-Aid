# Smart Writing Aid Frontend (React + Tailwind)

Polished, minimal React UI for Smart Writing Aid with mock data.

## Stack

- React (functional components)
- Tailwind CSS
- Vite

## Run

```bash
cd frontend-react
npm install
npm run dev
```

Open `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```

## Structure

- `src/components`: reusable UI components
- `src/pages`: page-level views
- `src/services`: mock API/data providers
- `src/hooks`: app hooks and timer logic
